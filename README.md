# ppp-loans-ds4a
Repo for the PPP Loan Data Warehouse.

## To Install and Run Jupyter Lab Locally:
### Windows:
```
python -m venv .venv  
.venv/Scripts/activate  
pip install -r requirements.txt  
jupyter-lab  
```

### Linux:
```
python3 -m venv .venv  
source .venv/bin/activate  
pip3 install -r requirements.txt  
jupyter-lab  
```
